# MLB Attendance Analytics ⚾

## Overview
This project analyzes the factors driving MLB attendance and provides recommendations on how the Cleveland Guardians can increase fan turnout.

## Key Question
What factors influence MLB attendance, and how can teams improve it?

## Data Sources
- Retrosheet (game attendance data)
- Spotrac (team payroll)
- Baseball Reference (team performance)

## Key Insights
- Higher payroll → higher win percentage
- Higher win percentage → higher attendance
- Teams cannot control opponents or schedule, but they *can* control roster investment

## Final Recommendation
Investing in player payroll is the most effective way to improve attendance through better team performance.

## Tools Used
- Python (Pandas, Plotly)
- Google Colab
